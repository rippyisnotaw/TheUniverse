About

Blocks turn into sand - clear pixels by lining up the same continous color from left to right.

Sandtrix is available for free on Desktop (Windows/Linux/MacOS) and as a web version.

Sandtrix+ is available on Steam. It extends the base game with new game modes, challenging color modes, and unlockable themes.
